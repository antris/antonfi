How to make a funnel in Google Analytics

February 27, 2015

So you know what a funnel is, but don’t know how to do it GA? I’ll show you the simplest possible funnel you can make!

Click here to view an example of a funnel
  1. Sign up and set up Google Analytics on your site.
  2. Make sure that for each step that you want in your funnel, you are sending a “page view” to Google Analytics. If you have a single-page app where the URL doesn’t change for each step, you can do this with the following JavaScript code: ga('send', 'pageview', <PAGE URL>, <PAGE TITLE>). For example ga('send', '/viewedProduct', 'Viewed product'). If your Google Analytics tracking code is set up for your whole site and you do a full page refresh on each step, you don’t need to edit the code at all. The title doesn’t really matter, but remember what the page URLs are. They will be needed later when you define the steps for the funnel.
  3. Sign in to Google analytics and find the site (or “Property” in Google Analytics lingo) that you want to set up the funnel for.
  4. Click the Admin tab on top.
  5. Click Goals on right.
  6. Click the red + NEW GOAL on top.
  7. In the Name field, you can put whatever you want. Usually it’s best to put a description of the “last step” in there. For example “Paid for purchase”. As for "Goal slot ID", I have no idea what it is. Leave it be.
  8. For Type, select Destination and press Next.
  9. In the Goal details section, for Destination put the URL that you sent to GA in step 2. This is the final step of the funnel.
  10. Keep "Value" off. You can set it up later.
  11. Put Funnel on!
  12. Now you'll set up the steps leading to the final step. For each step of the funnel, name the step descriptively, basically what the user did.
  13. Also, for each step put the URL exactly as you sent it to GA in step 2. Keep the "Required" switch in "No". I don’t know why it’s there.
  14. Press CREATE GOAL
  15. Deploy your code and wait for the results to come in!

Viewing your report

  1. When you think you have some results, go again to Google Analytics and find your site. This time, instead of the "Admin" tab, choose the Reporting tab on top of the site.
  2. In the top left search bar, type funnel and select Conversions > Goals > Funnel visualisation.
  3. In the Goal option drop down, make sure you have selected the Goal that you just set up.
  4. Scroll down.
  5. There it is.

If see your data for today, remember to put today in the selected date range (on top left). The data will show up with some delay.
